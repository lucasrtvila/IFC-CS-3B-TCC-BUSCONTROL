Seja Bem-Vindo ao repositório do app Bus Control.
Este app busca facilitar o trabalho de motoristas de transporte escolar particular por meio de um sistema que adapte as funções manuais de organização para funcionalidades digitais simples e automatizadas.
A primeira versão oficial já foi publicada em "Releases". Para testar, basta baixar o arquivo APK e instalar em seu dispositivo. Caso queira verificar o código fonte, este também está disponível.
